# Terminal2048

Created 10 June 2014 (Version 1.0)

Console Application of 2048 Game using **Ncurses**

## Installation

Ncurses:

```sh
sudo apt-get install libncurses5-dev
```

Use ```sh ncurses.h``` on your header file
